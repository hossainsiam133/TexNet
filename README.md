# TEXNET: A Novel Dataset for Handloom Fabric Classification Using Deep Learning

## Overview

TEXNET is an ongoing deep learning research project focused on the automated classification of authentic handloom fabrics and powerloom imitations using computer vision techniques.

The project introduces a curated fabric image dataset and investigates deep learning approaches for distinguishing traditional handloom weaving patterns from machine-produced alternatives. The primary motivation is to support textile authentication, cultural heritage preservation, and the protection of artisan communities.

## Problem Statement

Handloom fabrics play a significant role in preserving traditional craftsmanship and supporting rural economies. However, the increasing availability of powerloom-produced imitations creates challenges for consumers, retailers, and textile experts in verifying authenticity.

This project explores whether deep learning models can automatically identify visual characteristics that differentiate handloom fabrics from powerloom fabrics.

## Project Objectives

* Develop a high-quality image dataset for handloom and powerloom fabric classification.
* Investigate deep learning architectures for textile recognition.
* Evaluate transfer learning and custom CNN approaches.
* Build a foundation for future mobile-based fabric authentication systems.
* Contribute to research on computer vision applications in traditional textile industries.

## Dataset Highlights

* Images collected from authentic handloom and powerloom fabric samples.
* Multiple viewpoints and fabric regions captured under real-world conditions.
* Dataset preprocessing, augmentation, and quality validation performed for robust model training.
* Designed specifically for loom-type classification tasks.

> Note: The dataset is currently under active development and is not publicly released.

## Research Focus

Current investigations include:

* Custom CNN architectures
* VGG-based transfer learning models
* EfficientNet-based architectures
* Feature extraction and fine-tuning strategies
* Model generalization on real-world textile images
